使用 create table 语句可以创建表

SQL Server数据库的表由 元组和属性 组成

使用 delete 语句可以快速地删除表中的所有数据

SQL Server的表约束包括：

  • 每个数据库可以创建多少个表，没有限制
  • 每个表中最多定义1024列
  • 表和列的命名要遵守标识符命名规则
  • 列名在各自的表中必须唯一
  • 每列必须有一种并且只能有一种数据类型

having 子句的功能是指定组或聚合的搜索条件。having 子句通常与 group by 子句一起使用

表结构所包含的内容：表的名称，表的列，列的数据类型，列的空值，列的主键，列的索引，列的约束，列的默认值
